# Quick Calc CLI

A clean, beginner-friendly calculator CLI with safe expression evaluation.

## Usage
```bash
python3 main.py op add 12 3
python3 main.py op div 10 2
python3 main.py expr "(5 + 9) * 2"
```

## Features
- Basic operations (add/sub/mul/div)
- Input validation and division-by-zero protection
- Safe arithmetic expression mode via restricted AST parsing
