# Expense Flow Tracker

A portfolio-ready intermediate web app for personal expense tracking with budget visibility and category-level insights.

## Stack
- React + TypeScript + Vite
- Local Storage persistence
- Responsive CSS

## Features
- Add/delete expenses with validated input
- Monthly budget summary cards
- Category spending breakdown bars
- Recent expenses list sorted by date
- Local-only storage (no backend required)

## Run
```bash
npm install
npm run dev
```

## Future Improvements
- CSV export/import
- Monthly filter views
- Edit existing entries
- Receipt image attachments (local only)
